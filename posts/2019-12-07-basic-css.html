CSS 기본

  • auto -> 좌측, 우측

3개인 경우 (margin: 30px 10px 40px)

  • 첫번째 값 -> 위
  • 두번째 -> 좌, 우
  • 세번째 -> 아래

4개인 경우 (padding: 0 2opx 20px 20px;)

  • 첫번째 0 -> 상단
  • 두번째 20px -> 우측
  • 세번째 20px -> 하단
  • 네번째 20px -> 좌측
