CSS - font

구글의 font를 연결하고, 사용하는 방법을 알려준다

/*head에 아래 추가 */
<link href='http://fonts.googleapis.com/css?family=Alegreya:400italic,700italic,400,700' rel='stylesheet'
          type='text/css'>

현재 family로 Alegreya로 지정이 되어 있다. css에서는 아래와 같이 사용 한다.

h1, h2, h3, h4, h5, h6 {
    font-family: 'Alegreya';
}
