프로젝트 오일러를 시작하면서

원래는 4clojure를 다시 하고 싶었으나 사이트가 없어진 관계로 다른 것을 찾다가 수학도 볼 겸 해서 프로젝트 오일러를 다시 시작 했고 아래의 목적성을 가지고 있다

  1. 내 마음대로 내가 생각하는 방향대로 프로그램을 만든다

  2. 한 문제를 스스로 해결 후 더 나은 방법들을 찾아서 적용 해본다

  3. 함수형 방법으로 생각하고 프로그램을 만든다

위 3가지 목적으로 프로젝트 오일러를 시작 하게 되었고 지금도 풀어보는 중이다 좀 더 체계적으로 정리를 하기 위해서 블로그에 정리 한다
